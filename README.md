🎵 Ringtone Selection System
A robust, Java-based desktop application that simulates a mobile device's ringtone management interface. This project utilizes the Java Swing framework to provide a graphical user interface (GUI) where users can browse, preview, and set ringtones from a digital library.

📝 Project Description
This system is designed using Object-Oriented Programming (OOP) principles. It manages a collection of ringtone objects, each containing metadata such as name, genre, and duration. The application provides a seamless transition between the backend logic (Manager) and the frontend display (Swing UI), ensuring a responsive user experience.

Key Features
Dynamic Library: View a pre-loaded list of ringtones with detailed metadata.

Live Preview: "Play" a simulated preview of the selected tone via interactive dialogs.

Active Status Tracking: Real-time updates to the UI showing the currently active ringtone.

OOP Architecture: High-quality code structure using Encapsulation, Constructors, and Method Overloading.

Native Look & Feel: Automatically adapts to the operating system's UI style (Windows, Mac, or Linux).

🛠️ Requirements & Tech Stack
Prerequisites
To run this project, you need to have the following installed:

Java Development Kit (JDK): Version 8 or higher (Recommended: JDK 11, 17, or 21).

IDE (Optional): IntelliJ IDEA, Eclipse, NetBeans, or VS Code.

Git: For cloning the repository and version control.

Technologies Used
Language: Java

GUI Framework: Java Swing & AWT

Data Structure: ArrayList for dynamic storage

Design Pattern: Model-View-Controller (MVC) approach
